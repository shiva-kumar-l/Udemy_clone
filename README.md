# Udemy_clone
# 🎓 Udemy Clone - Responsive Front-End Website

A responsive static clone of the Udemy homepage built using **HTML5** and **CSS3**. This project demonstrates modern front-end development techniques, including semantic HTML, Flexbox, Grid, and responsive web design.

---

## 📌 Project Overview

The goal of this project is to recreate the look and feel of the Udemy homepage while practicing responsive web design principles and clean HTML structure.

This is a **static website** and does not include backend functionality or dynamic content.

---

## ✨ Features

### 🧭 Navigation Bar
- Udemy-style navigation
- Logo
- Search bar
- Navigation links
- Responsive layout

### 📂 Menu Bar
- Category menu similar to Udemy
- Easy navigation between sections

### 📚 Recommended Courses
- Course cards with:
  - Course image
  - Course title
  - Instructor name
  - Rating
  - Price

### 💡 Recommended Topics
- Topic suggestion section
- Input box for users to enter topics

### ⭐ Popular Courses
- Course listings
- Organized based on ratings and reviews
- Responsive card layout

### 📄 Footer
- Udemy-inspired footer
- Useful links
- Copyright information
